# Mini-Projet Cinéma ##

## Description ##
Ce projet est un site web permettant de consulter les films disponibles via l'API OMDb.  
L'objectif est de proposer aux clients un moyen simple de découvrir les films tendances, de rechercher des films en temps réel et de consulter les détails de chaque film.

## Pages ##
- **index.html** : Page d'accueil affichant les films, avec poster, titre, lien vers la page de détails et bouton "charger plus".  
- **search.html** : Page de recherche de films en temps réel (sans rechargement), avec affichage des résultats et bouton "charger plus".  
- **movie.html** : Page affichant les détails d'un film (titre, poster, résumé complet, genre, acteurs, notes et date de sortie DVD formatée en français).

## Outils utilisé ##
- HTML / CSS  
- JavaScript (fichiers modulaires, un fichier JS par page)  
- OMDb API  

## Installation et utilisation ##
1. Cloner le dépôt ou télécharger le ZIP du projet.  
2. Ouvrir le projet avec un serveur HTTP, par exemple :  
   - **VS Code Live Server**  
3. Naviguer entre les pages : `index.html`, `search.html`, `movie.html`.
